# 毫米波雷達 LilyGO T-Display-S3 顯示

## 簡介
本專案旨在使用 LilyGO T-Display-S3 顯示毫米波雷達的數據。此裝置結合了高效能的顯示屏和強大的處理能力，適合各種應用場景。

## 快速開始
1. **硬體準備**：
   - 確保已擁有 LilyGO T-Display-S3 開發板。
   - 連接毫米波雷達模組。

2. **原始韌體測試安裝**：
   - 使用 [官網韌體](https://espressif.github.io/esptool-js/) 進行固件燒錄。
   - 設置波特率為 38400。

3. **運行項目**：
   - 上傳代碼至開發板。
   - 啟動設備並觀察顯示屏上的數據。
